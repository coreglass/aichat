AI Chat Client

版本 0.1.0

一个现代化的AI聊天客户端，支持多种大模型和自定义endpoint。

主要功能

  • 支持多种AI模型（智谱AI、Gemini等）
  • 自定义API endpoint
  • 流式输出，实时显示响应
  • Markdown渲染支持
  • 代码高亮和一键复制
  • 对话历史管理
  • 翻译助手
  • 可自定义界面样式

技术栈

Tauri 2.x + Rust + Vite + Vanilla JavaScript
